# 科学气质测试

一个静态网页小测验。用户回答 24 道科学史风格的问题后，网页会生成 6 维科学气质雷达图，并给出对应的科学发现者类型。

## Features

- 24 道单选题
- 6 个雷达图维度：观察力、数学化、实验精神、怀疑精神、跨界想象、技术工具
- 每个维度大约有 4 道主要测量题，题目顺序混合呈现，避免按维度分段造成暗示
- 每道题的选项显示位置经过平衡轮换，减少总选同一位置造成的偏差
- 结果页显示 1-100 的典型性分数，用来表示类型轮廓是否集中
- 通过 Google Apps Script + Google Sheets 记录匿名完成次数和匿名测试结果
- 8 种结果类型，尽量覆盖不同学科传统：
  - 伽利略型实验观察者：物理天文
  - 达尔文型生命综合者：生命科学与演化
  - 居里型化学实验家：化学、材料与实验科学
  - 南丁格尔型数据医学者：医学、统计与公共健康
  - 图灵型计算建模者：计算科学与算法思维
  - 古道尔型行为田野者：行为科学与田野观察
  - 卡森型生态连接者：生态环境与系统风险
  - 波普尔型怀疑检验者：科学哲学与方法论
- 原生 JavaScript Canvas 雷达图，无需后端
- 可部署到 GitHub Pages

## Scoring

每个选项会给 1-2 个维度增加权重。完成答题后，网页先累加 6 个维度的原始分数，再归一化到 0-100，用于显示雷达图。结果类型根据每个类型对应的两个主导维度进行匹配，得分最高者作为最终科学气质类型。

典型性分数不是能力分，也不是智力分。它衡量的是结果轮廓的集中程度：最高维度和第二高维度差距越大、六个维度越不平均，典型性越高；如果六个维度都很高且彼此接近，则典型性较低，表示当前类型判断更像一个宽泛倾向。

## Anonymous Result Logging

网页通过 Google Apps Script Web App 将匿名测试结果写入 Google Sheets。记录字段包括类型、典型性分数和 6 个维度分数，不记录姓名、邮箱或 IP 地址。
