# 科学气质测试

一个静态网页小测验。用户回答 12 道科学史风格的问题后，网页会生成 6 维科学气质雷达图，并给出对应的科学发现者类型。

## Features

- 12 道单选题
- 6 个雷达图维度：观察力、数学化、实验精神、怀疑精神、跨界想象、技术工具
- 8 种结果类型，尽量覆盖不同学科传统：
  - 伽利略型实验观察者：物理天文
  - 达尔文型生命综合者：生命科学与演化
  - 居里型化学实验家：化学、材料与实验科学
  - 南丁格尔型数据医学者：医学、统计与公共健康
  - 图灵型计算建模者：计算科学与算法思维
  - 古道尔型行为田野者：行为科学与田野观察
  - 卡森型生态连接者：生态环境与系统风险
  - 波普尔型怀疑检验者：科学哲学与方法论
- 原生 JavaScript Canvas 雷达图，无需后端
- 可部署到 GitHub Pages

## Run Locally

直接打开 `index.html`，或启动本地静态服务器：

```bash
python -m http.server 8780
```
