# 科学气质测试

一个静态网页小测验：用户回答 12 个科学史风格的问题后，网页会生成
6 维科学气质雷达图，并给出对应的科学史人格类型。

## Features

- 12 道单选题
- 6 个雷达图维度：观察力、数学化、实验精神、怀疑精神、跨界想象、技术工具
- 8 种结果类型：伽利略型、牛顿型、达尔文型、居里型、图灵型、费曼型、麦克斯韦型、波普尔型
- 原生 JavaScript Canvas 雷达图，无需后端
- 可部署到 GitHub Pages

## Files

```text
index.html
```

## Run Locally

直接打开 `index.html`，或启动本地静态服务器：

```bash
python -m http.server 8780
```

## GitHub Description

一个科学史风格的趣味测评网页，回答问题后生成科学气质雷达图和科学发现者类型。
